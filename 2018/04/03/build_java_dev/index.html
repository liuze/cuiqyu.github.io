腾讯云搭建java开发环境

Apr 3, 2018 | 笔记liunx | 阅读

近期，在同学群里看到一条分享的参团链接，进去一看，腾讯云服务器在搞活动，很实惠（大概一年120元+赠送四个月）。想想觉得身为一名程序员服务器还是需要一个（可能有点装*的成分吧，哈哈！）于是乎我也就买了人生中第一台属于我自己的服务器。

其实一开始买过服务器之后，我都不知道可以来干嘛，然后linux又不怎么会，但是买都买了，就去找点开源项目放上去吧，不能浪费了不是。于是我就开始百度搭建开发环境，下面就记录我搭建环境的过程吧！

服务器安装centOS7.4系统

第一步：购买成功后进入腾讯云个人云主机后台：

点击重装系统，如图：选择你所需的linux操作系统，这里我选的是CenOS7.4，此处输入的密码就是你登录linux系统的密码，账户名默认root

点击开始重装后主机的监控状态将会显示重装中，稍等两分钟左右将会显示运行中，此时你的系统重装完成！

第二步：使用linux管理工具（我这里使用的是免安装的secureCRT管理工具，需要的可以去我百度云下载，密码：o9hj）点击快速连接如下图，如果进入右下角表示登录成功！

（注意：如果你登录过程中提示端口连接失败，那你需要去服务器后台检查22端口是否放通，默认是开通的！如图：这里我配置了一些常用的端口，可供参考！），此时，你的系统已经准备好，开始安装java开发环境吧！
