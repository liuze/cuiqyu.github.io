Java进阶之路

Apr 26, 2018 | java | 阅读

写在前面： 近段时间因为工作的原因，在考虑换一份工作；在面试几家后我自己的感觉就是知识很欠缺，很多知识点都不是很清楚，甚至于就完全不懂；我在想我也是快工作两年的程序员了，为什么我现在感觉都不如刚毕业的时候呢？我觉得是工作之后人变得懒惰了，变得社会了，开始选择安逸了，越来越没有上进心了，只是过着每天上班下班的生活，对未来依然一片茫然的。想想我觉得很可怕“我们不能再该奋斗的年代选择了安逸”。所以我准备开始我的java进阶之路了，想提高自己需要有方式方法，给自己定个目标。所以我从网上搜集整理了一些资料，初级程序员的进阶道路该怎么学。如有整理不妥当之处，还望大神指点，在此谢过了！

一、数据结构与算法

不管我们是用什么语言开发，都离不开数据结构与算法。

1.常用的数据结构：

数组、链表、堆、栈、队列、Hash表、二叉树……

2.算法的思想：

算法的时间复杂度与空间复杂度的分析与计算

算法的八大思想： 递推、递归、穷举、贪心、分治、动态规划、迭代、分支界限

3.经典算法：

八大排序算法：
直接插入排序、希尔排序、简单选择排序、堆排序、冒泡排序、快速排序、归并排序、基数排序

经典查找：
顺序查找、二分查找、二叉排序树查找

4.高级数据结构：

B+/B-树、红黑树、图……

二、java基础相关

万丈高楼平地起，只要基础牢，才能走的更远更稳。

1.java的四大特性：

抽象、封装、继承、多态

2.基础语法：

java的语法格式、常量变量、变量的作用域、方法及方法的重载、运算符、程序流程控制、八大基本类型（byte、short、int、long、float、double、boolean、char）及对应的包装类（Byte、Short、Integer、Long、Float、Double、Boolean、Character）

3.集合类：

Collection以及各种List、Set、Queue、Map的实现以及集成关系，实现原理

Collections和Arrays
